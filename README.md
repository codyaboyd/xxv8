# xxv8
Server/Client for building cross-platform executables from node.js projects using a REST API.
# Unix/Linx Instructions
./xxv8 (help)
./xxv8 server portNum (start server)
./xxv8 client nodeProject serverIP:port (send project for compilation. Put one level above your projects)
# Windows Instructions
xxv8.exe (help)
xxv8.exe server portNum (start server)
xxv8.exe client nodeProject serverIP:port (send project for compilation. Put one level above your projects)
